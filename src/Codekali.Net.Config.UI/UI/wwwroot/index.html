✦ Best Practices
⚠ Use IOptionsSnapshot<T> or IOptionsMonitor<T> — not plain IOptions<T> — so config changes take effect without a restart.
⚠ Never commit secrets to source control. Keep passwords, tokens, and API keys in environment-specific files excluded via .gitignore.
✓ Use appsettings.{Environment}.json for environment-specific overrides. Keep appsettings.json as the base with safe defaults only.
✓ Use colon-separated paths for nested keys — e.g. Logging:LogLevel:Default — when adding or editing in tree view.
✓ Create a 💾 Backup before making bulk changes. Backups are stored alongside your appsettings files as timestamped .bak files.
